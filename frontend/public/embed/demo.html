Merchant checkout demo

This static page embeds the Link2Pay "Pay with USDC" button. The merchant pre-creates a payment link (CRYPTO or Bre-B off-ramp) in the Link2Pay dashboard, then drops the button with that link id. Clicking it opens the hosted checkout in a popup — no API keys, nothing custodial.
